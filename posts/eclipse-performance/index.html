Dicas para melhorar a perfomance no Eclipse

3 Mar 2019 2 minutes read

Sempre execute o JDK e o Eclipse mais recentes

Na maioria das vezes, todas as novas versões do JDK e do Eclipse incluem correções e otimizações de versões anteriores. Certifique-se de estar usando uma versão de 64 bits do Eclipse e do Oracle JDK. Para desenvolvimento que seja web, é melhor usar o Eclipse para Java EE e não o Eclipse para Java.

Oracle JDK: http://www.oracle.com/technetwork/java/javase/downloads

Eclipse: https://eclipse.org/downloads/

Ajustes nas Preferências do Eclipse

  • General > Startup and Shutdown remova todos os plug-ins ativados na inicialização
  • General > Editors > Text Editors > Spelling Desativar verificação ortográfica
  • General > Validation > Suspend all Suspender todas as validações
  • Window > Customize Perspective Remova itens que você não usa ou deseja (as teclas de atalho são seus amigos)
  • Install/Update > Automatic Updates > Desmarque a opção “Automatically find new updates”
  • General > Appearance > Desmarque a opção “Enable Animations”

Em caso de problemas com atraso na resposta da digitação, você pode realizar as configurações abaixo e sempre que precissar de uma “assistência” ao programar, utilize o Ctrl + Espaço

  • Java > Editor > Content Assist > Desmarque a opção “Enable Auto Activation”
  • Advanced > Remover todos os tipos indesejados

Use seu teclado

Utilize os atalhos da ide ao seu favor! Uma das vantagens do Eclipse é a enorme quantidade de atalhos de teclado que ele oferece. Você pode remapear se estiver acostumado a outros esquemas.

  • Ctrl + N Criar um novo arquivo
  • Ctrl + Shift + R Abrir o código de um arquivo (recurso)
  • Ctrl + Shift + T Abrir o código de um tipo
  • Ctrl + : Pular para o próximo erro
  • Ctrl + Shift + G Procura por referências
  • Ctrl + Shift + P Selecione o par correspondente
  • Alt + setas Ir para frente / trás
  • Ctrl + Espaço Autocomplete
  • Ctrl + 3 Acesso rápido
  • Ctrl + Shift + F Formatar codigo-fonte
  • Ctrl + Shift + O Organiza as importações
  • Alt + Shift + R Renomear refatorando
  • Ctrl + D Excluir linha

Mais atalhos na documentação: https://www.eclipse.org/pdt/help/html/keymap.htm
